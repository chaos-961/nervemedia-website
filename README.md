# Nerve Media

Static landing page for Nerve Media, a Beirut marketing studio.

## Files

- `index.html` contains the page markup and content.
- `styles.css` contains the responsive layout, visual system, and animations.
- `app.js` contains scroll effects, reveal animations, and the background canvas.

## Local Preview

Open `index.html` in a browser, or serve the folder with any static file server.

## GitHub Pages

This repository includes a GitHub Actions workflow that publishes the static site
to GitHub Pages from the `main` branch.

After pushing to GitHub, enable Pages with **GitHub Actions** as the source in
the repository settings.
